== Description ==

Prime Testimonial Carousel is a lightweight, powerful plugin for displaying customer testimonials on your WordPress site. Choose from 6 professionally designed layouts and customize every aspect to match your brand.

= Key Features =

* **6 Beautiful Layouts** — Classic, Card, Modern, Minimal, Masonry, and Grid
* **Fully Responsive** — Looks perfect on desktop, tablet, and mobile
* **Gutenberg Block** — Native block editor support
* **Shortcode Support** — Use anywhere with `[prime_testimonial_carousel]`
* **Touch & Swipe** — Natural swipe gestures on mobile
* **Keyboard Accessible** — Full arrow-key navigation
* **Star Ratings** — Optional 1-5 star rating display
* **Author Photos** — Featured image support for each testimonial
* **Grouping** — Organize testimonials into categories
* **Zero Dependencies** — No jQuery, no third-party libraries on the frontend
* **Customizable Colors** — Match your site's branding
* **Autoplay Control** — Configurable timing with pause-on-hover
* **Translation Ready** — Full i18n support
* **RTL Support** — Works with right-to-left languages
* **Lightweight** — Minimal CSS and JS footprint

= Layout Options =

1. **Classic** — Clean bordered cards with subtle shadows
2. **Card** — Elevated cards with colored top accent
3. **Modern** — Gradient background with prominent styling
4. **Minimal** — Simple left-border design with italic text
5. **Masonry** — Pinterest-style variable-height grid
6. **Grid** — Equal-height CSS grid layout

= Accessibility =

* WAI-ARIA labels on all controls
* Keyboard navigation (arrow keys)
* Focus pause on autoplay
* Respects `prefers-reduced-motion`
* Semantic HTML structure

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/prime-testimonial-carousel/`, or install through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **Testimonials → Add New** to create your first testimonial.
4. Fill in the author name, title, company, rating, and content.
5. Set a featured image for the author photo.
6. Use the shortcode `[prime_testimonial_carousel]` in any post or page, or insert the Testimonial Carousel block.

== Frequently Asked Questions ==

= How do I display testimonials on my site? =

Use the shortcode `[prime_testimonial_carousel]` in any post, page, or widget. Alternatively, insert the Testimonial Carousel block in the Gutenberg editor.

= Can I filter testimonials by category? =

Yes. Create a group under **Testimonials → Groups**, assign testimonials to it, and use `[prime_testimonial_carousel group="your-group-slug"]`.

= Is the plugin mobile-friendly? =

Absolutely. The plugin automatically adjusts the number of visible columns based on screen size and supports native touch swipe gestures.

= Does it work with page builders? =

Yes. Since the plugin uses a standard shortcode, it works with Elementor, Beaver Builder, Divi, WPBakery, and any other page builder that supports shortcodes.

= Can I customize the colors? =

Yes, go to **Testimonials → Settings** to configure primary color, text color, background color, and border radius.

= Is it translation ready? =

Yes. The plugin is fully translation ready with a `.pot` file included in the `/languages` directory.
